# 👾 Portfólio Acadêmico: Weslley Eugênio - Neo-Retro

---

## 🎯 Sobre o Projeto

Este é o meu primeiro portfólio acadêmico e pessoal, desenvolvido para apresentar minhas habilidades e projetos na área de Desenvolvimento de Sistemas. O design é inspirado na estética **Neo-Retro** e **Pixel Art**, criando uma experiência única e temática.

O projeto foi construído com foco em fundamentos sólidos do Front-end: HTML, CSS e JavaScript puro, garantindo desempenho e total controle sobre o estilo.

## ✨ Destaques e Funcionalidades

* **Tema Exclusivo:** Paleta de cores inspirada em temas Neon e Cyberpunk, utilizando variáveis CSS.
* **Efeito Máquina de Escrever:** Implementado em JavaScript puro (`script.js`) na seção introdutória (`.my-area`), que simula a digitação de texto.
* **Navegação Suave:** Scroll suave configurado em CSS e complementado por JavaScript para um deslocamento otimizado entre as seções.
* **Design Responsivo:** Layout totalmente adaptável para telas grandes e pequenas (Mobile First), garantido pelas `media queries`.
* **Organização:** Código modularizado em três arquivos principais (`index.html`, `styles.css`, `script.js`) para fácil manutenção.

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |
| :--- | :--- |
| **HTML5** | Estrutura semântica e acessível do conteúdo. |
| **CSS3** | Estilização, layout Flexbox/Grid e criação dos efeitos Neon. |
| **JavaScript (Puro)** | Lógica de comportamento (Efeito Máquina de Escrever e Navegação). |

## 🔗 Acesso e Execução

### 💻 Execução Local

Para rodar o portfólio na sua máquina e inspecionar o código, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/Weslley-322/Porfolio.git
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd Portfolio
    ```
3.  **Abra no navegador:**
    Como é um projeto estático (HTML, CSS, JS puros), basta abrir o arquivo `index.html` em qualquer navegador.
